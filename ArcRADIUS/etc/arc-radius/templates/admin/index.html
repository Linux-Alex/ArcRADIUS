Welcome!

ArcRADIUS is a simple RADIUS server providing authentication and OTP (One-Time Password) options for secure access and remote control.

The name "ArcRADIUS" stands for Access and Remote Control RADIUS. It emphasizes the service's focus on providing secure access and control over remote resources.

Key features of ArcRADIUS include:

  • Centralized authentication for network access
  • Support for One-Time Passwords (OTP) for enhanced security
  • Easy integration with existing systems
  • Scalability to handle large numbers of users
  • Configurable access control policies

Whether you need to secure network access for your organization or provide remote access to resources, ArcRADIUS has you covered.

Get started with ArcRADIUS today and experience secure, reliable authentication and access control.

Installation Guide

Before installing ArcRADIUS, make sure to disable FreeRadius:

sudo systemctl stop freeradius.service
sudo systemctl disable freeradius.service

Follow these steps to install and configure ArcRADIUS:

  1. Configure Client Address in the Edit Address tab. The instructions for adding a new client are available on that page.
  2. Restart ArcRADIUS with the command: 
    sudo systemctl restart arc-radius.service
  3. Make a user account for the RADIUS client in the User tab.
  4. You can get the user's OTP QR code by clicking the Show QR code button.
