Add a new client entry

  1. The client name must be unique.
  2. The client configuration follows the YAML structure outlined below:
client_name:
  type_net: [subnet | ip]
  IP: [IP_address | subnet_mask]
  name: [descriptive_name]
  secret: [shared_secret]
  • client_name: The unique name/key of the client.
  • type_net: Specifies the type of network (subnet or IP).
  • IP: Specifies the client's IP address or subnet.
  • name: A descriptive name for the client.
  • secret: The shared secret used for authentication.

After changes you must restart the ArcRADIUS Server with:
sudo systemctl restart arc-radius.service
